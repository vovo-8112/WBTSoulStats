# WBTSoulStats

A Flutter-based web app that allows users to view and track WBT Soul statistics using public blockchain data. Designed for mobile-friendly use with the ability to add the app to your home screen on iPhone as a Progressive Web App (PWA).

## How to Use

Visit the app here: [https://vovo-8112.github.io/WBTSoulStats/](https://vovo-8112.github.io/WBTSoulStats/)

To install as a PWA on iPhone:
1. Open the link in Safari.
2. Tap the "Share" button.
3. Select "Add to Home Screen".

## Features

- Soul ID explorer integration
- Direct link to on-chain claim function
- Optimized for web and mobile
- Minimal dependencies


## How to Deploy

To deploy the app to GitHub Pages, run the deployment script from the project root:

```bash
bash lib/FlutterRunBuild.sh
```

This will:
- Clean the build
- Compile the Flutter Web app
- Push the build to the `gh-pages` branch for hosting
